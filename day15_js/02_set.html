자바스크립트에서 데이터 타입의 확장 개념 집합 데이터를 다룰 수 있는 객체

  • set은 중복데이터 허용x : 중복데이터가 들어오면 체크 후 하나만 남김
  • set() : 파라미터는 주로 배열에 사용, 배열의 index, value 값을 활용하여 집합데이터를 만듦
  • set 전용 메서드들이 있음.
  • new set() or new set(array) : set 객체를 생성 및 초기화
  • set.add(value) : 데이터 추가
  • set.delete(value) : 데이터 삭제
  • set.has(value) : 데이터의 존재 여부 확인 true / false 리턴
  • set.forEach(callback function) : set 순환
  • set.entries(), set.keys(), set.values() : 리턴 타입 => setIterator 형식 별도의 메서드를 활용하여 접근 (for of)
  • set.size : set의 개수 (괄호없음)
